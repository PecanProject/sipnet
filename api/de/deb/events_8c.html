Detailed Description

Handles reading, parsing, and storing agronomic events for SIPNET simulations.

The events.in file specifies agronomic events. See the input documentation (currently parameters.md) for information on the format of that file. Also, see test examples in tests/sipnet/test_events_infrastructure/ and tests/sipnet/test_events_types/.
