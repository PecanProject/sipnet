Detailed Description

Handles reading, parsing, and storing agronomic events for SIPNET simulations.

The events.in file specifies agronomic events. See the input documentation (currently parameters.md) for information on the format of that file. Also, see test examples in tests/sipnet/test_events_infrastructure/ and tests/sipnet/test_events_types/.

void processEvents ( void  )

Process events for current location/year/day.

For a given year and day (as determined by the global climate pointer), process all events listed in the global events pointer for the referenced location.

For each event, modify flux variables according to the model for that event, and write a row to events.out listing the modified variables and the delta applied.
