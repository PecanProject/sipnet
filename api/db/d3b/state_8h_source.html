379 // Re: rSoil vs maintRespiration
380 // When microbes are in effect, maintResp is the respiration term for the
381 // microbes, and rSoil is calculated taking maintResp into account (which
382 // seems fine).
383 // However, when microbes are off, maintResp and rSoil are treated as the
384 // same, which makes _some_ sense (and maintResp is calculated as
385 // heterotrophic soil resp is described in [1]), but the use in the code is
386 // confusing. Also, the description of maintResp (before this update) only
387 // mentions the microbe case, furthering the confusion. For microbes off, I
388 // would have expected maintResp to be 0, and rSoil to be calc'd as maintResp

390 // The GROWTH_RESP flag may also play in here, as when that is on, we split
