410 // Re: rSoil vs maintRespiration
411 // When microbes are in effect, maintResp is the respiration term for the
412 // microbes, and rSoil is calculated taking maintResp into account (which

415 // same, which makes _some_ sense (and maintResp is calculated as
416 // heterotrophic soil resp is described in [1]), but the use in the code is
