389 // Re: rSoil vs maintRespiration
390 // When microbes are in effect, maintResp is the respiration term for the
391 // microbes, and rSoil is calculated taking maintResp into account (which

393 // However, when microbes are off, maintResp and rSoil are treated as the
394 // same, which makes _some_ sense (and maintResp is calculated as
395 // heterotrophic soil resp is described in [1]), but the use in the code is
396 // confusing. Also, the description of maintResp (before this update) only
