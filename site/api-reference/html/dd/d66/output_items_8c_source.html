65/* Allocate space for a new outputItems structure, return a pointer to it
66 filenameBase is the base name of the files to which we'll output
67 - we'll output to <filenameBase>.<name> for each output item
68 separator is the character separating values in the output files (e.g. space, tab, or comma)
